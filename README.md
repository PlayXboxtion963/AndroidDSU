### 软件介绍

###### 功能

1. 支持了DSU协议中的所有按键
2. 支持手机陀螺仪与加速度计数据
3. 支持了DSU Unofficial的震动马达实现
4. 莫奈取色Android规范的UI
5. Android实体音量按键的支持
6. 双虚拟摇杆的支持
